Key Design Elements

  • API group clustering — CRDs grouped by their API group with instance counts
  • Green dot = CRD is healthy (operator running, API responding)
  • Instance count — number badge shows how many CRs of this type exist
  • Purple "Networking" badge = assigned to a domain section (appears as a tab there)
  • "Assign →" button = unassigned, click to assign to a domain
  • Click any card to drill into the resource list (same ResourceTable pattern)
